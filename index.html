Nombre del Estudiante:

Maryorie Viluce

Objetivo y Resumen del Trabajo:

Implementar un modelo de clasificación utilizando datos de clientes para predecir si abandonarán un servicio (churn). Se aplicaron técnicas básicas de aprendizaje supervisado.

Se cargó un archivo CSV con información de clientes. Se exploraron los datos, se graficó la distribución de clases y se entrenó un modelo de Árbol de Decisión. Finalmente, se evaluó su precisión.

Lenguaje y herramientas utilizadas:

  • Python (Thonny)
  • Librerías: pandas, matplotlib, scikit-learn

Resultado del modelo:

  • Modelo: Árbol de Decisión (DecisionTreeClassifier)
  • Precisión alcanzada: 94.8%
  • Variable objetivo: churn

Conclusión:

El laboratorio permitió aplicar un flujo completo de trabajo de IA: preparación de datos, entrenamiento de modelos y evaluación. Usar Thonny en vez de Jupyter no afectó los resultados esperados.
